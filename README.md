🐾 Animal Classification System
This project leverages Deep Learning to identify and classify animal species from images. The goal is to build a high-accuracy model capable of handling diverse lighting conditions and camera angles.
Tech Stack
Language: Python
Framework: PyTorch
Data Processing: NumPy, Pandas, OpenCV

📸 Demo & Visuals
Dataset Classes
The model is trained to recognize the following 10 categories:

🦋 Butterfly | 🐱 Cat | 🐔 Chicken | 🐄 Cow | 🐶 Dog

🐘 Elephant | 🐴 Horse | 🐑 Sheep | 🕷️ Spider | 🐿️ Squirrel
